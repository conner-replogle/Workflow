# Running the Project

## Dependencies

- [Install Go](https://go.dev/doc/install)
- [Install Node.js](https://nodejs.org/en/)

## First Launch

### App

1. Navigate to the `workflow` folder.
2. Install dependencies:
    ```bash
    npm install
    ```
3. Run the app:
    ```bash
    npm run start
    ```

### Server

1. Navigate to the `server/` directory.
2. Start the server:
    ```bash
    go run cmd/main.go
    ```
